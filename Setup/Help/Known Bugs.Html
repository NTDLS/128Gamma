Known Bugs

128Gamma would not of been released if there were any "known" bugs.

Windows 95 Kernel Does Not Have Sufficient Threading Support To Run 128Gamma versions above 3.0B, Therefore 128Gamma 3.0B and up "WILL NOT" Run On A Windows 95 Platform.

If you have found error in 128Gamma and were not running another encryption program available from NetWorkDLS please see "Reporting Bugs" under the "Contact" section in this help file.
