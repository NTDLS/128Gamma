Encrypting & Decrypting Single Files

To encrypt a single file simply right click on any file and select "Open With 128Gamma" from the context menu. 128Gamma will then prompt you for a password to encrypt the file with. Enter a password and click ok.

To decrypt an Single Files, Simply open it (By hitting enter or double clicking on its icon). 128Gamma will prompt you for the correct password. Enter a password and click ok.
