Reporting Bugs

128Gamma has been tested on Windows 98, Me, NT 4.0, & W2k.

Windows 95 Kernel Does Not Have Sufficient Threading Support To Run 128Gamma versions above 3.0B, Therefore 128Gamma 3.0B and up "WILL NOT" Run On A Windows 95 Platform.

Do Not Run Any Other Encryption Program Published By NetWorkDLS Along Side 128Gamma, Doing so will cause a temporary file conflict.

In the past releases of 128Gamma few bugs have been reported but all found have been fixed. If 128Gamma crashes and gives you an error, The error message should be followed by an error code (Error: xx). Please send the error code, a detailed description of what you were doing when the error occurred, what operating system you are using to: Info@NetWorkDLS.Com

Also, if you notice an error or a system malfunction that seems to be originating from 128Gamma please send a detailed description of the problem and why you think its originating from 128Gamma to: Info@NetWorkDLS.Com

Please include you name in the e-mail so we can address you respectfully.

Allow atleast 24 - 72 hours for a response.
