Encrypting & Decrypting Entire Folders

Encrypted Packages are files that contain 2 or more files that have been encrypted.

To create an Encrypted Package simply right click on any folder and select "Open With 128Gamma" from the context menu. 128Gamma will then prompt you for a password to encrypt the Package with. Enter a password and click ok.

To decrypt an Encrypted Package, Simply open it (By hitting enter or double clicking on its icon). 128Gamma will prompt you for the correct password. Enter a password and click ok.
